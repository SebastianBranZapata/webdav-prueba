# Usage

> Upload file to Salesforce Commerce Cloud instance with WebDav

## Install node packages

``` bash
npm install
```

## Requirements

* node >= 10

## Configuration

```json
{
  "hostname": "dev01-mysandbox.demandware.net",
  "client_id": "aaaaaaaaaaaaaaaaaaaaaa",
  "client_secret": "mysupersecretpassword"
}
```

## Run

``` bash
node .\index.js
```
